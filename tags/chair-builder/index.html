Mesh FlattenTools

StatusCompletedDate2012-12-06

Small JS Library to flatten meshes

A small javascript library built on top of THREE.js to flatten any mesh into a 2D plane. This was a core component of the Kammy Chair Builder webapp, open-sourced as I thought it might be of some use to others.
